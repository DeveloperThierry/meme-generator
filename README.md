### Project Overview

This is a fun, client-side application built with **React** that allows users to create custom memes. 

It fetches a list of popular meme templates and enables users to add, position, and style **top and bottom text captions** before downloading the final image.

---

### Key Features âœ¨

* **Template Fetching:** Automatically pulls a dynamic list of current meme templates from a public API.
* **Real-time Editing:** Instantly preview your meme captions as you type.
* **Custom Text:** Add **top and bottom captions**, with options for font size, color, and positioning.
* **Downloadable Output:** Export the finished meme as a high-quality image (JPEG/PNG).
* **Responsive Design:** Works great on desktop and mobile devices.
